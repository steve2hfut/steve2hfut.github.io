一些简单的实列练习

Markdown 标题

Markdown 标题有两种格式。

1.使用 = 和 - 标记一级和二级标题
测试（用```成对来显示代码块）

2.使用 # 号可表示 1-6 级标题，
一级标题对应一个 # 号，
二级标题对应两个 # 号，最多可到六级。

Markdown 段落

1.段落的换行是使用两个以上空格加上回车。

测试

世界

2.使用空行来重启一个段落。（用这个方法显示时会空一行）
